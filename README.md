# ecoleta

### Aplicação para cadastro e consulta de pontos de coleta de reciclados.

**Página web desenvolvida na semana de estudos Next Level Week realizada pela [Rocketseat](https://rocketseat.com.br/)

### Aplicaçã feita com:
* HTML
* CSS
* JavaScript
* Express
* Nunjucks
* SQLite3
